# 指定教材单元批量上传 (Course Unit Uploader)

## 技能描述 (Description)
这个技能用于自动化处理本地文件夹中的视频文件。它会在教学系统的“自制教材”中搜索特定的教材名称（支持模糊与多关键词匹配），在指定的教材内新建以【本地文件夹名】命名的单元，并将该文件夹内的视频上传到该单元中。
支持“全部文件夹批量上传”和“精准上传单个指定文件夹”。

## 适用场景 (When to use)
1. 当用户要求“把 [某文件夹] 传到 [某教材项目] 中”时（例如：请帮我将4月文件夹中的视频上传到CBBC 2026中），提取目标教材名和文件夹名作为参数传入。
2. 当用户要求“把所有视频作为单元追加到 [某教材项目]”时，仅提取目标教材名并传入。

## 运行环境与依赖 (Requirements)
1. 依赖 Python 3.8+ 环境。
2. 运行前需要执行 `pip install -r requirements.txt` 和 `playwright install chromium` 安装环境。
3. 首次使用前需引导用户手动运行 `python auth_setup.py` 生成 `auth.json` 登录凭证。

## 执行指令 (Execution Command)
此脚本强制需要 `-c` (--course) 参数来定位目标教材。

1. **上传特定文件夹到特定教材（最常用场景）**：
   如果用户说“将4月文件夹传到CBBC 2026”，则执行：
   `python main.py -c "CBBC 2026" -f "4月"`

2. **将本地所有文件夹作为单元上传到特定教材**：
   如果用户说“把本地所有动画都传到测试教材”，则执行：
   `python main.py -c "测试教材"`

## 预期结果 (Expected Result)
脚本将自动拉起浏览器，搜索教材、新建对应名称的单元、注入本地文件并完成暂存操作。完成后控制台会打印“任务执行完毕”。
